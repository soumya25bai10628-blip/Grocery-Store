# Grocery-Store
This project is a simple Python-based Grocery Store Management System that allows users to add, display, search, update, delete, and purchase grocery items. All data is stored in a text file, making it easy to manage small inventory systems through a user-friendly menu-driven program.

Overview
A simple menu-driven Python project that helps manage a grocery store inventory.
Users can add items, display all items, search, update, delete, and purchase items.
All data is stored in a text file (grocery_data.txt), making it easy to handle small-scale inventory systems.

Features
Add new grocery items
Display all stored items
Search for an item by name
Update item price or quantity
Delete an item
Purchase items with automatic billing
Stores data in a text file

Technologies / Tools Used
Python 3
File Handling (Text File Storage)

Basic concepts: loops, functions, conditions, string operations

Steps to Install & Run the Project
Download or copy the Python file (PYTHON PROJECT.py).
Make sure the file grocery_data.txt exists (it will be auto-created when adding items).
Open a terminal/command prompt.
Navigate to the project folder:
cd your-folder-name

Run the program:
python "PYTHON PROJECT.py"

Instructions for Testing
Run the program and try all menu options from 1 to 7.
Add a few items first so that Display, Search, Update, Delete, and Billing work correctly.
Enter invalid choices or misspelled item names to check error handling.
Try purchasing items with different quantities to test stock validation.
